# MERN Stack Blog App — Full Tutorial Guide

Build a full-stack blog application using **React 19**, **Express 5**, **MongoDB**, and **Clerk** authentication — complete with infinite scroll, user roles, image optimization, and one-click VPS deployment.

<Callout type="info">
    This guide is based on a full video tutorial. All source code is available on GitHub (see the repo link in the original video). Fork the starter branch to match the exact React 19 RC version used here.
</Callout>

---

## What You'll Build

- 🏠 **Homepage** — featured posts, category filters, animated CTA
- 📜 **Post list** — infinite scroll, search, sort filters, category sidebar
- 📄 **Single post** — rich content, sticky sidebar, save/delete actions
- ✍️ **Write page** — rich text editor, image/video upload, cover photo
- 🔐 **Auth pages** — Clerk-powered login & register
- 👮 **Admin role** — feature posts, delete any post or comment
- 🚀 **VPS deployment** — Hostinger + CloudPanel + pm2

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 19, Vite, Tailwind CSS, React Router v6 |
| Backend | Express 5, Node.js |
| Database | MongoDB Atlas + Mongoose |
| Auth | Clerk |
| Media | ImageKit |
| Data fetching | TanStack React Query v5 |
| Editor | React Quill New |
| Deployment | Hostinger VPS, CloudPanel, pm2 |
