# URLs / Routes?

A **URL (route)** tells Django what to do when a user visits a specific link.

## Common URL Examples

| URL | What It Shows |
|-----|---------------|
| `/` | Landing page |
| `/posts` | List of all posts |
| `/posts/1` | A specific post |
| `/login` | Login page |
| `/profile` | User profile page |

> **Think of URLs as addresses** — each one points Django to a specific destination in your app.
