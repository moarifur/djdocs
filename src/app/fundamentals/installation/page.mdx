# Django Installation

Follow each step carefully. **Do not skip any step.**

---

## Step-by-Step Setup

### Step 1 — Check Python Installation

```bash
python -V
```

**Expected Output:** `Python 3.x.x`

> If you see a version number, Python is installed correctly.
> If you see `"python is not recognized"` or `"Python was not found"`, Python is **not** installed.
> **Fix:** Open Microsoft Store → search "Python" → click Install. Then restart your terminal and run `python -V` again.

---

### Step 2 — Upgrade pip

```bash
python -m pip install --upgrade pip
```

> `pip` is the package manager used to install Django and other libraries.

---

### Step 3 — Create Virtual Environment

```bash
python -m venv venv
```

> This creates a folder named `venv` in your project directory.

---

### Step 4 — Activate Virtual Environment

```bash
venv\Scripts\activate
```

**Expected Output:** `(venv) C:\your-path>`

> The `(venv)` prefix confirms you are inside the virtual environment.

---

### Step 5 — Install Django

```bash
pip install django
```

> Django installs **only inside this virtual environment** — not globally.

---

### Step 6 — Verify Installation

```bash
pip freeze
```

**Expected Output:** `Django==5.x.x`

> This lists all packages installed in the current virtual environment.

---

### Step 7 — Create Django Project

```bash
django-admin startproject myproject
```

> This generates your Django project folder and configuration files.

---

### Step 8 — Move into Project Folder

```bash
cd myproject
```

> You must be inside the project folder before running the server.

---

### Step 9 — Run Development Server

```bash
python manage.py runserver
```

**Expected Output:** `Starting development server at http://127.0.0.1:8000/`

> Open your browser and go to [http://127.0.0.1:8000/](http://127.0.0.1:8000/) to see the Django welcome page!
